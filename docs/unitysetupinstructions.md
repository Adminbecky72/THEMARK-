# unity setup instructions for Rapture Survival Game

## scene setup
1.  **create tags**
2.  "player'
3.  "enemy'
4.  "hiding spot'
5.  "safe zone'

6. **Player Gameobject**
7. -add a 'rigidbody2d' (set body type to dynamic)
8. -ADD A BOXCOLLIDER2D OR 'CIRCLECOLLIDER2D'
ATTACH:
   -PLAYERMOVEMENT.CS
   -PLAYERHEALTH.CS
   -PLAYERSTEALTH;CS
